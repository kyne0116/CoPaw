# 京东购物助手 Skill

## 功能说明

自动打开浏览器访问京东商城，根据关键词搜索商品，提取价格信息并推荐性价比最高的商品。

## 使用方法

在 CoPaw 对话中直接说：

```
帮我在京东搜索"机械键盘"并推荐最便宜的
```

或者：

```
京东上搜索"无线鼠标"，比较前5个商品的价格
```

## 工作流程

1. 启动浏览器（非无头模式，可见操作过程）
2. 访问 jd.com
3. 在搜索框输入关键词
4. 点击搜索按钮
5. 提取商品标题、价格、店铺、链接
6. 分析并推荐价格最低的商品
7. 关闭浏览器

## 返回结果

```json
{
  "keyword": "搜索关键词",
  "products": [...],
  "recommendation": {最便宜的商品},
  "total": 5
}
```
